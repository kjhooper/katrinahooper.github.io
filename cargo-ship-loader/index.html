Algorithms & Simulation

Manifest

A physics-grounded optimisation engine for packing 20 ft and 40 ft ISO containers into a configurable cargo hull — Coastal Feeder, Handymax, or Panamax. Seven solver strategies, from a greedy baseline to a hybrid that combines RL–Bayesian warm start with Simulated Annealing refinement, all competing on the same four-axis balance metric under a hard weight constraint.

Hull Geometry

Ships are modelled as 3-D voxel grids. The hull widens tier by tier from a narrow keel up to full beam — matching real cargo-hold geometry. Three pre-built classes are available (Coastal Feeder 12×9×5, Handymax 24×11×7, Panamax 36×13×9) alongside a fully custom configurator. Hull cells are never valid placements, and a hard weight limit rejects any container that would push the ship over its maximum cargo capacity.

The Seven Solvers

All solvers share the same BaseSolver interface and report a unified final_score() metric, making them drop-in replacements for one another.

Greedy Heuristic

Single-pass, heaviest-first. Scores every valid position in O(1) using running sums for five balance terms (stability height, trim, list, diagonal, stacking bonus). Fastest solver; already achieves > 99.5 % balance.

Beam Search Heuristic

Maintains the top-K partial solutions at each placement step. K = 1 is identical to greedy; K = 5–10 broadens the search without exponential blowup. Deterministic and interpretable.

Simulated Annealing Heuristic

Warm-starts from the greedy solution and iteratively relocates “top” containers (nothing stacked above). Accepts worse moves early on to escape local optima; cools geometrically to commit to the best-seen state.

Bayesian Opt ML

Treats the five scoring weights as hyperparameters and uses a Gaussian-Process surrogate (optuna TPE) to find the combination that maximises balance quality. Each GP trial is a full greedy run (< 1 s), so 50 trials take under a minute.

Neural Ranker ML

Replaces the hand-crafted scoring function with a two-layer MLP (128×128 hidden units). Trained via behavioural cloning from Beam Search: for each placement decision, it learns which of the valid positions the best beam would have chosen.

RL Bayesian ML

Predicts the optimal CargoLoader scoring weights directly from manifest statistics using an MLP pre-trained on BayesOpt solutions (imitation learning), then fine-tuned via reward-weighted regression with sequential importance resampling. Near-instantaneous at inference.

RL Bayesian + SA ML Heuristic

Uses RL Bayesian as a warm start, then applies Simulated Annealing refinement on the result. Combines fast, globally balanced initial placement from the pre-trained model with SA’s local search to escape the residual imbalances that ML alone leaves under tight weight or space constraints.
